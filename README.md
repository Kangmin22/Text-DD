⚔️ DND TEXT RPG: THE MECHANICS

파이썬 기반의 텍스트 RPG 엔진 프로젝트입니다. v9.0 수학 엔진과 유연한 상태 머신 아키텍처를 기반으로 설계되었습니다.

🚀 주요 기능

수학 엔진 (v9.0): WoW 방어 공식과 PoE 공격 버킷 시스템이 결합된 하이브리드 전투 계산기.

상태 머신 (FSM): Title, Town, Combat 등 유연한 상태 전환 시스템.

아이템 & 장비: 데이터 기반의 아이템 시스템 및 실시간 스탯 반영 로직.

다이나믹 로그: 전투 상황에 따른 화려한 플레이버 텍스트 및 시각적 효과.

📂 프로젝트 구조

src/core/: 엔진 및 상태 머신 코어

src/models/: Actor, Item 등 데이터 모델

src/systems/: 전투, 성장, 인벤토리 등 로직 시스템

src/utils/: 데이터 로더 및 유틸리티

src/data/: 게임 밸런스 데이터 (JSON)

🛠 실행 방법

python src/main.py
